HR Analytics Dashboard
A Power BI dashboard presenting key HR insights based on employee, department, and salary data.

📊 Overview
This project analyzes:

- Workforce distribution

- Active vs terminated employees

- Salary by department

- Salary by gender

- Salary by years of service

The goal is to provide a clear, business‑ready HR overview.

📁 Data Model
The dashboard uses four tables:

- Employees

- Departments

- Absences

- DimDate

Relationships were built to ensure correct filtering and aggregation.

📈 Visuals Included
Total Employees (KPI)

- Employee Count by Department

- Active vs Terminated Employees

- Average Salary by Department

- Average Salary by Gender

- Average Salary by Years of Service

🛠 Tools
Power BI Desktop

- Power Query

- DAX

- Star Schema Modeling
